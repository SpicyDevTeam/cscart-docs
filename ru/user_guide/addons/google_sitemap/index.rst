*******************************
Карта сайта для Yandex и Google
*******************************

Этот модуль генерирует адаптированную под веб-сканеры карту вашего сайта в формате XML. Он значительно упрощает и улучшает индексацию сайта, повышает рейтинг вашего сайта при поиске через Google и Yandex. Это особенно полезно, если в магазине есть большое количество страниц товаров и категорий в каталоге.

Модуль позволяет настроить каждый элементв карте сайта: домашнюю страницу, страницы товаров и категорий, отдельные страницы, а также страницы характеристик с типом "бренд".

У модуля **Карта сайта для Yandex и Google** есть следующие настройки:

* **Включать в карту сайта (sitemap)** — если настройка включена, то элемент будет добавлен в карту сайта.

* **Частота обновления** — интервал времени между двумя обновлениями элемента.

* **Приоритет** — приоритет элемента относительно других элементов карты сайта.
